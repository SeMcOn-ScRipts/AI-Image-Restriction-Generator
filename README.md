# AI-Image-Restriction-Generator
AI tool that turns keywords into detailed scene descriptions for creative writing and AI image prompts. Example: “gunsmithing” → rich visual scene of the action. It helps generate clearer, vivid prompt outputs for models and storytelling tasks by transforming short inputs into rich descriptive scenes for AI models. Useful for prompt engineering 

Step 1 :
Download the libraries in requirements.txt file .

Step 2 :
Add your Gemini Api-Key from " https://aistudio.google.com/app/api-keys " to the script "image.py" in line 4 .

Step 3 : 
To run the Python file you need to run CMD in the folder that have the file "image.py" and run " image.py " command .

Step 4 :
To use the tool just write the act you want to describe for example gunsmithing and it will give you a output then just go to your AI Agent preferably Gemini and give it this prompt " I want you to generate a image about the charchter ( then give it your charchter for example: Superman ) and follow this act in the charchter "then add the paragraph the tool gave you" .
